# IPL-DASHBOARD-
📌 Project Overview
This project presents an interactive IPL (Indian Premier League) Dashboard built in Power BI, analyzing data from 2008 to 2025.
The goal of this project is to transform raw cricket data into meaningful insights, enabling users to explore trends in team performance, player statistics, and match dynamics through a visually engaging interface.
🚀 Key Insights & Features
📊 Overall KPIs
Total Matches Played
Total Runs Scored
Total Wickets Taken
Overall Strike Rate
🏆 Team Performance
Comparison of top-performing teams
Consistency across seasons
Total runs contribution by each team
🧑‍💻 Player Performance
Top run scorers across IPL history
Comparative analysis of key players
Contribution to overall scoring trends
📈 Seasonal Trends
Runs scored per season
Identification of high-scoring eras
Year-on-year performance patterns
⏱ Match Phase Analysis
Powerplay performance
Middle overs stability
Death overs acceleration
🥇 Historical Winners
Season-wise IPL champions
Dominance trends over the years
🛠 Tools & Technologies
Power BI – Dashboard development & visualization
DAX (Data Analysis Expressions) – Calculations & measures
Data Cleaning & Transformation – Handling missing values & structuring data
🎯 Project Objectives
Build a clean and interactive dashboard for sports analytics
Apply data modeling and DAX for real-world datasets
Practice storytelling through data visualization
Create a portfolio-ready project for recruiters
