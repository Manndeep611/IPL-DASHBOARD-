# IPL-DASHBOARD-
# 🏏 IPL Data Analytics Dashboard (2008–2024)

## 📌 Overview
An interactive **Power BI dashboard** analyzing IPL data from 2008 to 2024.  
This project focuses on transforming raw cricket data into meaningful insights around **team performance, player statistics, and match trends**.

---

## 🚀 Features
- 📊 KPI Summary: Matches, Runs, Wickets, Strike Rate  
- 🏆 Team Performance Analysis  
- 🧑‍💻 Player Performance Insights  
- 📈 Season-wise Trends  
- ⏱ Phase Analysis (Powerplay, Middle, Death Overs)  
- 🥇 Historical IPL Winners  

---

## 🛠 Tech Stack
- **Power BI**  
- **DAX (Data Analysis Expressions)**  
- **Data Cleaning & Transformation**
